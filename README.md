# desertification patterns prediction

this project is about analyzing climate data and ndvi to understand vegetation changes.

## data used
- nasa climate data (rain, temperature, soil moisture)
- ndvi data

## steps
- cleaned and merged the climate data
- converted data into monthly format
- combined it with ndvi values
- trained a random forest model

## results
- r2 score around 0.69
- rmse around 0.047

## tools used
- python
- pandas
- matplotlib
- sklearn

## output
- ndvi trend graph
- feature importance graph
